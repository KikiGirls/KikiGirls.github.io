games_Lec 10~12 - 几何 Geometry 隐式与显式 曲线与曲面

Author: 王思懿 Date: October 30, 2024  0:23:28 Category: games101

几何的表达方式

隐式表达

  • 基于 点的分类
  • 采样可能较难
  • 内外测试简单

种类

  • 代数曲面 (Algebraic Surface)

    • 最直接的，用数学公式
    • 不直观

  • 构造实体几何 (Constructive Solid Geometry, CSG)

    • 基本形状的布尔操作组合成复杂形状
    • 通过布尔运算结合隐式几何

  • 距离函数 (Distance Functions)

    • 相关概念：带符号的距离场 (Signed Distance Field)
    • 解析形式表达

  • 水平集 (Level Sets)

    • 用网格的方式描述距离
    • 例子：CT扫描

  • 分形 (Fractals)

    • 自相似性
    • 递归定义

  • …

优点

  • 紧凑的描述方式（如用一个函数）
  • 某些查询简单（如对象内外部、到表面的距离）
  • 适用于光线-表面交点计算（稍后详细讨论）
  • 对于简单形状，提供精确描述 / 无采样误差
  • 易于处理拓扑结构变化（如流体）

缺点

  • 难以表示复杂形状

Explicit 表达

  • 所有点都直接给出或通过参数映射给出
  • 采样简单
  • 内外测试较难

种类

  • 点云 (Point Cloud)

    • 一堆点，可以表示任意几何
    • 适合超大数据集（远超每像素一个点）
    • 常转换为多边形网格
    • 在欠采样区域难以绘制

  • 三角形/多边形网格 (Triangle/Polygon Mesh)

    • 存储顶点和多边形（通常为三角形或四边形）
    • 数据结构较复杂
    • 图形学中最常用的表示方式之一
    • 例子：.obj 格式
      • 一种文本文件格式，指定顶点、法线、纹理坐标及其连接关系

  • 细分曲面、NURBS

  • 贝塞尔曲面 (Bezier Surfaces)

  • 细分曲面 (Subdivision Surfaces)

  • NURBS

注意

没有“最佳”表达方式，每种选择都最适合不同的任务或几何类型

在计算机图形学中还有更多的隐式表示方法








曲线 Curve

Bézier Curves 贝塞尔曲线

  • 仿射变换前后统一
  • 凸包性质：形成的曲线一定在控制点形成的凸包内

高阶贝塞尔曲线的控制难题

高阶贝塞尔曲线较难控制，因为任何一个控制点的移动都会影响整个曲线的形状。

改进方案：分段贝塞尔曲线 (Piecewise Bézier Curves)

  • 通过拼接多个低阶贝塞尔曲线来简化控制
  • 例如：分段三次贝塞尔曲线 (Piecewise Cubic Bézier)
    • 每段由四个控制点定义

光滑性保证

  • 保证曲线的光滑性（切线连续）：
    • 在连接处，内部控制点的前后切线点需要共线，以避免切线的突变

参考

  • 贝塞尔曲线编辑

连续性的定义

  • C^0 连续：无断点
  • C^1 连续：无突变

样条曲线 (Spline)

样条是一种连续曲线，其构造方式使其能够通过给定的一组点，并且拥有一定数量的连续导数。（一种受控的曲线）

B样条 (B-splines)

  • 基函数样条 (Basis Splines)
    • 以伯恩斯坦多项式 (Bernstein Polynomial) 作为基函数
    • 满足局部性，即改变一个控制点只影响局部区域
    • 是图形学中较复杂的一部分
    • 可以视为贝塞尔曲线的超集

拓展：B样条和NURBS

  • B样条（B-splines）
  • NURBS（非均匀有理B样条）

进一步学习：B样条和NURBS简介

曲面

贝塞尔曲面 (Bézier Surfaces)

贝塞尔曲面通过两个不同的参数 ( t ) 和 ( u, v ) 控制：

  • 采用 4x4 个控制点，形成四条贝塞尔曲线，每条包含 4 个控制点
  • 在相同参数 ( u ) 处取四个控制点，再通过参数 ( v ) 确定曲面上的最终点

网格 (Mesh)

更广泛使用的是网格表示法。

网格操作：几何处理 (Geometry Processing)

  • 网格细分 (Mesh Subdivision) - 上采样
    • 提高分辨率
  • 网格简化 (Mesh Simplification) - 下采样
    • 降低分辨率
    • 尽量保持形状和外观
  • 网格正规化 (Mesh Regularization)
    • 避免出现特别奇怪的三角形
    • 调整采样分布以提高网格质量

Loop细分 (Loop Subdivision)

细分的一个典型应用场景是**位移贴图 (Displacement Mapping)**，需要模型足够细致，因此需要细分（最好是动态细分）。

Loop细分的“Loop”指的是发明者的名字，与循环无关。

要求

  • 适用于三角形网格 (Triangle Mesh)

步骤

  1. 创建更多的三角形（顶点）

    • 将每个三角形分割为四个较小的三角形

  2. 调整顶点位置（形状需要发生改变）

    • 根据权重分配新的顶点位置
    • 新增的顶点和原有的顶点以不同的方式更新

Catmull-Clark细分 (Catmull-Clark Subdivision)

适用于通用网格 (General Mesh)

术语

  • **非四边形面 (Non-quad face)**：非四边形的面
  • **奇异点 (Extraordinary Vertex)**：指度数不等于4的点（degree ≠ 4）

每次细分步骤

  1. 在每个面中添加一个顶点
  2. 在每条边的中点处添加一个顶点
  3. 将所有新顶点连接

奇异点在第一次细分增加[非四边形面数量]，后续不变；所有非四边形面在第一次细分都会消失

网格简化 (Mesh Simplification)

目标：在保持整体形状的前提下减少网格元素的数量

应用场景

  • 移动端渲染
  • 远距离视角 (如LOD系统)

几何的层次结构

**边收缩 (Edge Collapse)**：合并顶点

  • 哪些边需要合并？如何合并？
    • 使用**二次误差度量 (Quadric Error Metrics)**，将顶点放置在使误差之和最小的位置

使用二次误差进行简化

  • 提出者：Garland & Heckbert, 1997
  • 方法：迭代地合并具有最小分数的边
    • 合并一条边会影响其他边，需要更新数据结构
    • 使用优先队列或堆来组织边
  • 特点：贪心算法，非全局最优，但能有效降低误差
